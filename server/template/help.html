1. Login to new node, run:

$ mkdir /var/stathub
$ cd /var/stathub
on 32 bits node
$ wget http://{{.server}}/static/client_x86 -O client
on 64 bits node
$ wget http://{{.server}}/static/client_x86_64 -O client
 $ chmod +x client
$ ./client

2. Setup config, input:

> Please enter the NAME of THIS node [vagrant]:
(Press ENTER or input a name)
> Please enter the URL of SERVER :
http://{{.server}}
> Please enter the KEY of SERVER :
{{.key}}

3. Setup crontab, run:

$ (crontab -l; echo "* * * * * cd /var/stathub; ./client >>/var/stathub/client.log 2>&1") | crontab -
